# Pokémon media

Put artwork and cries here. Paths are referenced from `data/pokemon/*.json`.

```
public/assets/pokemon/
  0025-pikachu.png
  0025-pikachu.ogg
```

JSON:

```json
{
  "image": "0025-pikachu.png",
  "cry": "0025-pikachu.ogg"
}
```

Served on Pages as `/aetherium-pokedex/assets/pokemon/...`.
